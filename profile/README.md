# SellRepo

Sell access to your GitHub repositories.

Deploy with [hatchbox.io](https://hatchbox.io)

## Created by GoRails

[GoRails](https://gorails.com) is the leading Ruby on Rails education platform. We built SellRepo as an example Rails application with payments and licenses to help people sell their own code repositories and serve as an educational code base to learn from.
